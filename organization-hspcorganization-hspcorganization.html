StructureDefinition: HSPCOrganization

The official URL for this profile is: 

http://hl7.org/fhir/StructureDefinition/organization-hspcorganization-hspcorganization

The "Organization" model describes an organization. A Provider Organization is a role played by an instance of this organization.
