This is a value set defined by the FHIR project.

Summary

Defining URL:http://hl7.org/fhir/ValueSet/diet-type
Name:Diet Codes
Definition:DietCode : Codes that can be used to indicate the type of food being ordered for a patient. This value set includes all children of SNOMED CT Concepts (US Extension and Core) from SCTID 182922004 Dietary regime (Therapeutic or Preventive Procedure) and is provided as a suggestive example
