Formal value Set definition: XML or JSON

1.25.4.1.19398.1 ActInvoiceGroupCode

Type of invoice element that is used to assist in describing an Invoice that is either submitted for adjudication or for which is returned on adjudication results. Invoice elements of this type signify a grouping of one or more children (detail) invoice elements. They do not have intrinsic costing associated with them, but merely reflect the sum of all costing for it's immediate children invoice elements. (OID = 2.16.840.1.113883.1.11.19398)
