4.5.11 FHIR Core Goal Profile

This profile holds core Goal extensions

4.5.11.1 Content

Extensions:
goal-categoryCode describing the type of goal :

The type of goal, expressed as a code, for example, Goal: weight loss, [SNOMED CT: 289169006].

goal-targetTarget for goal :

The element describing the target for the goal, allowing the progress of the goal to be monitored against an observation.

goal-reasonRejectedThe reason the goal was not accepted :

The reason the goal was not accepted. Applies only if the status of the goal is rejected.

goal-relationshipGoals related to this one :

Establishes a relationship between this goal and other goals.
