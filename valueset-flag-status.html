1.25.2.1.2001 Value Set http://hl7.org/fhir/ValueSet/flag-status

This is a value set defined by the FHIR project.

Summary

Defining URL:http://hl7.org/fhir/ValueSet/flag-status
Name:FlagStatus
Definition:Indicates whether this flag is active and needs to be displayed to a user, or whether it is no longer needed or entered in error
OID:2.16.840.1.113883.4.642.2.2001 (for OID based terminology systems)

1.25.2.1.2001.1 Content Logical Definition

This value set has an inline code system http://hl7.org/fhir/flag-status, which defines the following codes:

CodeDisplayDefinition
active ActiveA current flag that should be displayed to a user. A system may use the category to determine which roles should view the flag
inactive InactiveThe flag does not need to be displayed any more
entered-in-error Entered in ErrorThe flag was added in error, and should no longer be displayed

 

This value set is used in the following places:

  • Flag.status (Required)
