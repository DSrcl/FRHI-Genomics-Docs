4.30.5 Resource Sequence - Detailed Descriptions

Detailed Descriptions for the elements in the Sequence resource.

Sequence
Definition

This resource records information about sequence of a moleculde. E.g. a DNA sequence.

Control1..1
InvariantsDefined on this element
tla-1: Start and end position should be non-negative, and end position must be larger than start position. (xpath: boolean(f:start/@value >= 0) and boolean(f:start/@value >=0) and boolean(f:end/@value > f:start/@value))
tla-2: CIGAR string should conform with extended-cigar's syntax. (xpath: matches(f:cigar/@value, '\*|([0-9]+[MIDNSHPX=])+'))
Sequence.chromosome
Definition

Chromosome of the sequence. E.g. "1" for chromosome 1. Note that value of chromosome is not limited to a human's chromosomes, as the sequence can be from other species such as Tuberculosis.

Control1..1
Typestring
Sequence.startPosition
Definition

0-based location start position (inclusive) of the sequence.

Control1..1
Typeinteger
Sequence.endPosition
Definition

0-based end location (exclusive) of the sequence.

Control1..1
Typeinteger
Sequence.direction
Definition

Direction of the strand. Forward means the sequence is in 5'-3' direction, reverse otherwise.

Control0..1
BindingStrandDirection: Direction of a sequence (Required)
Typecode
Sequence.cigar
Definition

Extended CIGAR string for aligning the sequence with reference bases. See detailed documentation here.

Control0..1
Typestring
Sequence.sample
Definition

Tissue sample from which molecules encoded by the sequence are extracted.

Control0..1
BindingSampleType: Type of the sample used to determine a seauence (Required)
Typecode
Sequence.lab
Definition

Lab where the sequence is determined.

Control0..1
TypeReference(DiagnosticOrder)
Sequence.patient
Definition

Whom the sequence is determined for. In a prenatal sequencing test, the patient being referenced is the birthgiver.

Control1..1
TypeReference(Patient)
Sequence.species
Definition

Species of the organism whom the sequence belongs to. The species is not limited to Homo sapiens. E.g. the sequence can be a DNA sequence of a bacteria.

Control1..1
BindingSNOMED CT Codes for species: Species of the organism to whom the sequence belongs to (Required)
TypeCodeableConcept
Sequence.quantity
Definition

Quantity of molecule encoded by the sequence presenting in the sample.

Control0..1
TypeQuantity
Sequence.genomeBuild
Definition

Genome assembly used to align the sequence.

Control0..1
Typestring
Sequence.type
Definition

Type of the sequence. E.g. DNA.

Control1..1
BindingSequenceType: Type of a sequence (Required)
Typecode
Sequence.referenceSequence
Definition

Reference bases of the sequence.

Control0..1
Typestring
Sequence.observedSequence
Definition

Observed bases of the sequence. Ploidy is represented by number of observedSequence.

Control1..*
Typestring
Sequence.name
Definition

Alterantive name of the sequence -- e.g. gene covered by the sequence, exome region in which the sequence is located, rsid of the sequence if it's a SNP, etc.

Control0..*
TypeCodeableConcept
